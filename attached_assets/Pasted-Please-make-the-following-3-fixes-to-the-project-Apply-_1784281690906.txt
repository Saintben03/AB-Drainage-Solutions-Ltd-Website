## FIX 2: Dynamic favicon per division domain

Since all 3 division sites are served from the same build (single-page app), we use a small inline script in `index.html` to swap the favicon before React loads, based on which domain is being visited.

### Step A — Create 3 new SVG favicon files in `artifacts/ab-drainage/public/`:

Create `artifacts/ab-drainage/public/favicon-construction.svg` with this content:
```svg
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100">
  <circle cx="50" cy="50" r="50" fill="#dc2626"/>
  <text x="50" y="67" font-family="Arial Black, Arial, sans-serif" font-size="44" font-weight="900" text-anchor="middle" fill="white">AB</text>
</svg>
```

Create `artifacts/ab-drainage/public/favicon-fencing.svg` with this content:
```svg
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100">
  <circle cx="50" cy="50" r="50" fill="#16a34a"/>
  <text x="50" y="67" font-family="Arial Black, Arial, sans-serif" font-size="44" font-weight="900" text-anchor="middle" fill="white">AB</text>
</svg>
```

Create `artifacts/ab-drainage/public/favicon-facilities.svg` with this content:
```svg
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100">
  <circle cx="50" cy="50" r="50" fill="#64748b"/>
  <text x="50" y="67" font-family="Arial Black, Arial, sans-serif" font-size="44" font-weight="900" text-anchor="middle" fill="white">AB</text>
</svg>
```

### Step B — Add the favicon-switching script to `artifacts/ab-drainage/index.html`

In `index.html`, find the Favicon comment block:
```html
    <!-- Favicon -->
    <link rel="icon" href="/favicon.ico?v=7" sizes="any" />
    <link rel="icon" type="image/svg+xml" href="/favicon.svg?v=7" />
    <link rel="icon" type="image/png" sizes="48x48" href="/favicon-48.png?v=7" />
    <link rel="icon" type="image/png" sizes="32x32" href="/favicon-32.png?v=7" />
    <link rel="icon" type="image/png" sizes="16x16" href="/favicon-16.png?v=7" />
    <link rel="apple-touch-icon" sizes="192x192" href="/favicon-192.png?v=7" />
```

Replace it with:
```html
    <!-- Favicon — default for abdrainage.co.uk -->
    <link id="favicon-main" rel="icon" href="/favicon.ico?v=7" sizes="any" />
    <link id="favicon-svg" rel="icon" type="image/svg+xml" href="/favicon.svg?v=7" />
    <link rel="icon" type="image/png" sizes="48x48" href="/favicon-48.png?v=7" />
    <link rel="icon" type="image/png" sizes="32x32" href="/favicon-32.png?v=7" />
    <link rel="icon" type="image/png" sizes="16x16" href="/favicon-16.png?v=7" />
    <link rel="apple-touch-icon" sizes="192x192" href="/favicon-192.png?v=7" />
    <!-- Division favicon switcher — runs before React mounts -->
    <script>
      (function() {
        var h = window.location.hostname;
        var svg = null;
        if (h.includes('abconstructionsolutions')) svg = '/favicon-construction.svg';
        else if (h.includes('abfencingsolutions')) svg = '/favicon-fencing.svg';
        else if (h.includes('abfacilitiesmgmt')) svg = '/favicon-facilities.svg';
        if (svg) {
          document.getElementById('favicon-main').setAttribute('href', svg);
          document.getElementById('favicon-svg').setAttribute('href', svg);
        }
      })();
    </script>
```
